# Sayılar 🐥

#### Numbers \( Sayılar \)

```javascript
let n = 175 // Tamsayı olabilir, int
n = 17.535 // Ondalıklı sayı olabilir, floating point
```

Sayı tipi integer ve floating point sayılarını temsil eder.

{% tabs %}
{% tab title="Çarpma" %}
```javascript
const a = 17;
const b = 5;

const total = a * b;

console.log(total); // 85
console.log(typeof total); // number
```
{% endtab %}

{% tab title="Bölme" %}
